Hoope

Hoope means "Redwood" in Ohlone language, native to the San Francisco Bay Area.
This website is a result of my passion for the game of football and the beauty of the Bay Area.

More detail:

Following technologies were used in the creation of this website: HTML, CSS, Javascript, Express, Typescript, Anguar, MongoDB, Mongoose.

Features

This website allows users to register and create player profiles. They can search for football pitches using Google Maps API, rate them and read reviews.

Future work

In the future, this website would add the functionality for people to create events and have people sign up for matches at pitches. Furthermore, users would be able to directly message others, rate them as players and organize into teams.
